Technical Architecture

Frontend Framework

  • React 18.2.0 with TypeScript
  • React Router 6.x for navigation
  • Tailwind CSS 3.x for styling
  • Vite as build tool

Key Files

  • src/App.tsx: Main application with routing
  • src/main.tsx: Application entry point
  • src/index.css: Global styles
  • src/components/: Reusable UI components
  • src/pages/: Page components
  • src/utils/: Utility functions (emailService.ts, googleDocs.ts)
  • src/assets/: Images and static assets

Styling Guidelines

  • Color Palette:
    • Primary: Amber (#b45309, #92400e)
    • Text: Stone (#1c1917, #78716c)
    • Background: Stone (#fafaf9)
    • Accents: White (#ffffff)
  • Typography:
    • Headings: Sans-serif, bold, text-4xl to text-5xl
    • Subheadings: Sans-serif, semibold, text-2xl to text-3xl
    • Body: Sans-serif, regular, text-base to text-lg
  • Components:
    • Buttons: Rounded corners, amber background
    • Cards: White background, subtle shadows
    • Sections: Consistent padding (py-12 to py-20)

Animation and Interaction

  • Typing effect on Home page
  • Scroll-triggered animations
  • Hover effects on buttons and links
  • Smooth page transitions

Content Management

  • Publication Updates: Add new publications to src/pages/Publications.tsx
  • CV Updates: Modify src/pages/CV.tsx
  • Research Projects: Update src/pages/Research.tsx
  • Image Assets: Add to src/assets/

Maintenance Guidelines

Regular Updates

  • Keep publications list current with new articles and books
  • Update CV as new positions, awards, or qualifications are acquired
  • Refresh research projects with new initiatives and completed work
  • Update statistics on Home page to reflect current numbers
  • Check and update all links to ensure they remain functional
  • Review and update content at least quarterly

Technical Maintenance

  • Keep dependencies updated through regular npm updates
  • Test across browsers (Chrome, Firefox, Safari, Edge) and devices (desktop, tablet, mobile)
  • Optimize images for web to maintain performance
  • Monitor site performance and make adjustments as needed
  • Implement security updates promptly
  • Maintain backups of the codebase

Development Workflow

  1. Local Development:
    • Install dependencies: npm install
    • Start development server: npm run dev
    • Access at http://localhost:5173/
  2. Code Quality:
    • Run linting: npm run lint
    • Fix linting issues: npm run lint:fix
  3. Building:
    • Create production build: npm run build
    • Preview production build: npm run preview
  4. Deployment:
    • Prepare for deployment: npm run predeploy
    • Deploy to hosting: npm run deploy
